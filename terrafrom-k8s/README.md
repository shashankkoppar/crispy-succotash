# GKE Cluster provisioning using terraform


# Requirements

1 - Terraform must be installed

2 - gcloud must be installed


# Step-by-step

1 - Checkout this Repo

5 - run terraform init (just once)

6 - terraform plan

7 - terraform apply

8 - go grab a coffee and wait for the automation magic! :)
